# Escargot

1 x big snail shell.
3 x neopixel inside snail shell
1 x glass jar with correct size (cf sketchup model).
Printed parts (cf sketchup model).
1 + Attiny85
1 potentiometer.
1 small rocker switch.

1 x proto board double sided.
